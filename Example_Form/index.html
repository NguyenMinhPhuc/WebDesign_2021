Attribute -- Thuộc tính của tag Form

Action Attribute

Chỉ định nơi nhận giá trị (file server side)

Method Attribute

Chỉ định cái phương gửi data (get, post)

Notes on Get

  1. data sẽ được thêm lên url theo cặp : Name/value
  2. không dung get khi gửi data bảo mật
  3. chiều dài của URL cố định 2048
  4. bookmark data
  5. get dùng cho data không cần bảo mật

Notes on Post

  1. them data vào body của request
  2. không giới dung lượng
  3. không bookmark data
